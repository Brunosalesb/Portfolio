Experiência

No ano de 2019, tive a oportunidade de participar de um projeto durante o último semestre de Análise e desenvolvimento de sistemas, um projeto que foi passado para nós de uma grande empresa brasileira de consultoria da área de TI, um projeto que consistia em um sistema intranet, com a função de gerenciar equipes para projetos, sistema de vagas dentro da empresa, cadastro de novos funcionários com seus dados pessoais e habilidades técnicas e comportamentais dentre outras funcionalidades.

O projeto foi desenvolvido por uma equipe de 5 pessoas durante 1 semestre, se baseave em um sistema monolítico, as tecnologias utilizadas foram C#, Angular.js, Sql Server, EntityFramework, AutoMapper, Odata, dentre outras tecnologias, além de também ter sido utilizado plataformas como Trello, Azure DevOps e Jira para Monitorar e Controlar o desenvolvimento das sprints do projeto.
